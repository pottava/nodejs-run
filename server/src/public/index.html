APIs

  • クライアント バージョン
    • GET /api/v1/versions
    • 例: curl -s http://localhost:8080/api/v1/versions | jq .
  • メトリクスの登録
    • POST /api/v1/metrics/:id
    • 例: curl -isXPOST -H 'Content-Type: application/json' -d '{"key":1,"value":"0.1"}' http://localhost:8080/api/v1/metrics/001
  • BigQuery のみ
    • POST /api/v1/big-query/:id
    • 例: curl -isXPOST -H 'Content-Type: application/json' -d '{"key":1,"value":"0.1"}' http://localhost:8080/api/v1/big-query/001
  • 外部 API 実行のみ
    • POST /api/v1/external-apis/:id
    • 例: curl -isXPOST -H 'Content-Type: application/json' -d '{"key":1,"value":"0.1"}' http://localhost:8080/api/v1/external-apis/001
